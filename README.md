# 半导体能带交互式仿真工具

一个基于 HTML/JavaScript 的纯前端仿真程序，用于直观展示半导体能带结构、费米能级位置及载流子浓度随材料、掺杂浓度、温度的变化。支持简并区修正和质量作用定律验证。

## ✨ 主要功能

- **多材料选择**：支持硅 (Si)、锗 (Ge)、砷化镓 (GaAs)，每种材料具有独立的禁带宽度和状态密度参数。
- **掺杂类型**：n 型、p 型、本征半导体。
- **动态参数调节**：
  - 掺杂浓度：对数范围 \(10^{12} \sim 10^{19} \, \text{cm}^{-3}\)（滑块 + 数值输入框）。
  - 温度范围：5 K ~ 500 K（滑块 + 数值输入框）。
- **实时能带图**：
  - 蓝色实线：导带底 (Ec)
  - 红色实线：价带顶 (Ev，设为 0 eV)
  - 绿色虚线：费米能级 (Ef)
- **载流子浓度信息**：
  - 电子浓度 \(n\)、空穴浓度 \(p\)
  - 质量作用定律乘积 \(n \cdot p\) 与本征载流子浓度平方 \(n_i^2\)
- **简并区处理**：
  - 当 Ef 进入导带或价带时，自动显示警告。
  - 简并区载流子浓度采用线性修正因子，使 \(n \cdot p \neq n_i^2\)，反映质量作用定律失效。
- **数值输入双向联动**：可在输入框中直接输入浓度或温度（支持科学计数法），按回车后滑块和图表同步更新。
- **左侧信息面板**：实时显示所有计算数值及禁带宽度 \(E_g\)。

## 🚀 使用方法

1. 将提供的 `band_diagram_simulator.html` 文件保存到本地。
2. 使用现代浏览器（Chrome, Firefox, Edge 等）打开该文件。
3. 通过左侧面板调整参数：
   - 选择材料（硅/锗/砷化镓）
   - 选择掺杂类型（n型/p型/本征）
   - 拖动浓度/温度滑块，或直接在下方输入框中输入数值并按回车
4. 观察右侧能带图的变化，以及左侧信息面板中载流子浓度的实时更新。
5. 当费米能级超出能带范围时，会显示“简并区”警告，此时质量作用定律不再成立。

## 📁 文件说明

- `band_simulator.html`：唯一的主程序文件，包含所有样式、布局和 JavaScript 代码。
- 无需任何外部依赖或服务器，纯本地运行。

## 🧪 技术特点

- **纯前端实现**：无需 Python 或其他后端环境，开箱即用。
- **实时响应**：所有控件均绑定事件，调整后立即刷新图表和数值。
- **物理模型**：
  - 玻尔兹曼统计（非简并近似）。
  - 温度依赖的有效状态密度 \(N_c(T), N_v(T) \propto T^{3/2}\)。
  - 本征载流子浓度 \(n_i(T) = \sqrt{N_c N_v} \exp(-E_g/(2kT))\)。
  - 费米能级通过二分法求解电中性方程。
- **简并区示意**：当 Ef 进入能带时，修正载流子浓度，体现简并效应。
- **质量作用定律验证**：非简并区 \(n \cdot p = n_i^2\)，简并区两者分离。
- **交互友好**：滑块与数值输入框双向绑定，支持科学计数法输入。

## ⚠️ 注意事项

- 简并区的载流子浓度修正仅为线性近似，用于定性演示，精确计算需使用费米-狄拉克积分。
- 材料参数基于 300 K 参考值，未考虑禁带宽度随温度的变化（可后续扩展）。
- 掺杂浓度范围固定，超过边界值会自动钳位。
